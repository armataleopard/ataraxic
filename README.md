# ATARAXIC ($calm) Website

A therapy-themed memecoin website for the ATARAXIC ($calm) project.

## Overview

This website showcases the ATARAXIC ($calm) memecoin with a hilarious therapy-themed design. It features ghostly animations, offbeat irony, and a mix of cartoon and eerie serenity styles.

## Features

- Responsive design for all device sizes
- Interactive elements and animations
- Randomly appearing ghost emojis
- Custom cursor effect on buttons
- Floating ghost animations
- Social media integration for Telegram and Twitter

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling and animations
- `script.js` - JavaScript for interactive elements
- Images:
  - `logo.jpg` - Main logo
  - `banner.png` - Hero section background
  - `1.png`, `2.png`, `3.png`, `4.png` - Ghost therapy session images

## Usage

Simply open `index.html` in a web browser to view the website.

## Sections

1. **Header** - Logo and branding
2. **Hero Banner** - Main call to action and buttons
3. **Ghost Session Slides** - Three therapy-themed slides
4. **Tokenomics** - "CALMonomics" section with project details
5. **Narrative** - Short story explaining why the project exists
6. **Footer** - Social links and disclaimer
